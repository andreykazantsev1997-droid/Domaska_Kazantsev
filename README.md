# Домашка 10.1

## Цель проекта 
Проект заключается в разработке виджета для банковских операций клиента

## Инструкция по установке и использованию функций
Данный проект содержит в себе три функции. Каждая из них отвечает за вывод определенной информации
для пользователя.

### Установка проекта:

1. Клонируйте репозиторий:
```
git clone https://github.com
cd domashka
```
2. Установите зависимости через Poetry:
```
poetry install
```
3. Активируйте виртуальное окружение:
```
poetry shell
```

Функция ***masks.py***

Данная функция выполняет маскировку номера банковской карты и счета пользователя.

Функция ***widget.py***

Данная функция выполняет импорт функции ***masks.py***, 
и добавляет подробную информацию о том что именно вводит пользователь. Учитывая название карты (Visa, MasterCard и т.д), 
а так же выводит дату в формате *"ДД.ММ.ГГГГ"*.

#### Как использовать функцию: 

При вызове функции необходимо ввести карту, номер счет и дату (в такой последовательности).

Пример вывода:

`Maestro 1596 83** **** 5199`

`Счет **9589`

`11.03.2024`

Функция ***processing.py***

Данная функция выполняет проверку введенных данных на соответсвие, а так же выполняет сортировку 
этих данных по дате (по убыванию).

#### Как использовать функцию: 

При вызове функции, она уже должна работать с заранее введенными списками словорей. 
Функция отсортирует и проверит списки словорей на соответствие. 

Пример вывода:

`[{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, 
{'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'}]`

Функция ***generators.py***

Данная функция выполняет сортировку списков по введенным наименованиям валюты ("USD","RUS" и т.д)
Так же функция возвращает список информации об транзакционных операций и выдает номер банковской карты в 
заданном диапазоне.

#### Как использовать функцию: 

При вызове функции, она уже должна работать с заранее введенным списком словарей. 
Функция выполняет сортировку списков по введенным наименованиям валюты,вернет информацию о транзакции
и так же по заранее введенным диапазоном чисел от 0000 0000 0000 0001 до 9999 9999 9999 9999. 
Выведет номер банковской карты.

## Тестирование
Проект покрыт тестами с использованием библиотеки pytest. 
Тесты проверяют логику маскировки карт/счетов и функции обработки транзакций.

**Как запустить тесты:**

1. Убедитесь, что у вас установлены зависимости: 
````
pip install pytest pytest-cov
````
2. Запустите тесты из корня проекта: 
````
pytest
````
3. Для получения отчета о покрытии тестами (coverage) используйте:
````
pytest --cov=src --cov-report=term-missing
````
